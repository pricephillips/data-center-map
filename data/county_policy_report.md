- Frame: 3144 counties. Puerto Rico municipios (n = 78) are excluded from the model and retained in the aggregate layer: no presidential margin exists and land-use institutions differ.
- Positive: county has at least one enacted restrictive policy, n = 195 (base rate 6.2 pct).
- The outcome is an enacted public record, which reduces but does not remove detection-bias exposure relative to opposition-presence outcomes.

## Validation

- Repeated stratified CV (5 folds x 5 repeats, seed 7).
- Specification selected by fixed criterion (see metrics JSON for the full grid): spec 'no_shift' with C = 0.01. Tier 3 variables are strongly collinear (density x population +0.87, income x education +0.69), so sign stability requires shrinkage; only sign-stable specifications within 0.01 AUC of the grid best are admissible, and among those the criterion retains as many variables as possible. Under strong shrinkage, coefficient magnitudes are compressed toward zero by design; read relative ordering and sign, not absolute size.
- ROC AUC: 0.82 (p10 0.76, p90 0.85).
- Brier: 0.053 vs base-rate Brier 0.058.
- Raw-score calibration slope 1.16 (shrinkage compresses raw scores); the model therefore includes a cross-fitted recalibration layer. Post-recalibration slope 0.99, intercept -0.02. Use calibrated_score from the scores file.

## Trust tiers

- Tier 1: political margin, existing DC count. Low detection-bias exposure; safest for external use.
- Tier 3: population, density, income, education. These variables also predict news coverage; on this public-record outcome the exposure is reduced, and it is still reported.

## Standing caveats

- Scores are out-of-fold: no county is scored by a model that saw it in training.
- The score is a resemblance measure over county profiles, calibrated to the observed enactment base rate. It is not a forecast of opposition to any specific project.
- Inputs are QC-gated upstream by county_aggregator.py; the input manifest is embedded in county_policy_metrics.json for reproducibility.
